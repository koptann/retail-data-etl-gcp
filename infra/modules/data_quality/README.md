# Data Quality Module - Validation & Testing

Creates data quality validation and monitoring infrastructure.

## Purpose

Implements comprehensive data quality checks throughout the pipeline.

## Resources Created (Phase 3)

- Cloud Functions for CSV validation
- Cloud Scheduler for daily quality checks
- BigQuery views for quality metrics
- Anomaly detection models

## Features

- ✅ **Pre-load Validation** - Catch bad data early
- ✅ **Quality Metrics** - Continuous monitoring
- ✅ **Anomaly Detection** - ML-based pattern detection
- ✅ **Automated Testing** - dbt test integration
