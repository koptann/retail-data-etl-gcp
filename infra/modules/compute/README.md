# Compute Module - Cloud Run Jobs & Artifact Registry

Creates compute resources for running dbt transformations in a serverless environment.

## Purpose

Manages Cloud Run jobs and container registry for the data transformation layer.

## Resources Created

- Artifact Registry repository for dbt images
- Cloud Run job for dbt execution
- Container configuration with secrets and environment variables

## Features

- ✅ **Serverless** - Pay only for execution time
- ✅ **Auto-scaling** - Handles variable workloads
- ✅ **Secret Management** - Secure credential mounting
- ✅ **Resource Optimization** - Right-sized CPU/memory
